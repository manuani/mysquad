# Scripts

Repo automation. Shell scripts and small Node programs that support the
build and the developer workflow.

## Current scripts

- `create_workspace_files.sh` — generator used to scaffold the initial
  service and package workspaces. Kept for reference; not part of the
  build.

## Adding a script

Make scripts executable (`chmod +x`) and document them here. If a script
becomes load-bearing for CI, also reference it from the relevant
`.github/workflows/*.yml`.
