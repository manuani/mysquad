## Layout

- `terraform/` — Infrastructure-as-Code for staging and production. Populated
  in Sprint 1.1.3 (Staging deployment pipeline). Cloud provider (AWS or GCP)

  Neo4j, and Redis. Populated in Sprint 1.1.2 (Local development environment).
